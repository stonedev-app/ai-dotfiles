## Git / GitHub CLI
### 参照系（自由に使ってよい）
- `git log` / `git diff` / `git status` / `git show`
- `gh`の`list` / `view` / `diff`系コマンド全般

### 書き込み系（issueのみ許可）
- `gh issue create` / `gh issue edit` / `gh issue close` / `gh issue reopen` / `gh issue comment`

それ以外のGit・gh操作はいかなる場合も実行しない。
特にPRの作成・マージ、push、releaseの作成、workflowの実行は絶対に行わない。
